# demo-flatlogic2

### To run project on local machine run

#### 1. Make sure you are using following node version

```
& node -v
> v20.11.1
```

#### 2. Install dependencies

```
& npm i
```

#### 3. Run project on localhost

```
 npm run dev
```

#### 4. Open project in browser

```
 http://localhost:8080/
```
